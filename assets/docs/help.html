Controls

To rotate the view, use the left mouse button and drag. To zoom in/out, use the mouse wheel.

Properties

You can write any valid math expression to the MathScript properties. Operators +, -, *, / have the usual meaning and ^ means exponentiation.

Builtin variables:

NameDescription
timeAn always increasing counter
randRandom value between 0 and 1
rand1Random value between -1 and 1
lifelineGoes from 0 to 1 in the lifetime of the particle
